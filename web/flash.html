How to flash

  1. Connect your ESP32-S3 board to this computer via USB.
  2. Enter bootloader mode: hold the BOOT button, press and release RESET, then release BOOT.
  3. Click the Install button for your board below.
  4. Select the serial port when prompted by the browser (look for a USB or JTAG device).
  5. Wait for the flash to complete. The board will reboot automatically.
