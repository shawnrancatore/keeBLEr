1. Click the Ultimate button to open the C64 menu
2. Navigate to F2 → WiFi settings
3. Select your keeBLEr network name
4. Enter the password you set above
5. The C64 will connect and get an IP (usually 192.168.4.2)
6. Enter that IP in the C64 panel and click Connect

The C64 Ultimate does not expose WiFi network selection via its HTTP API — this must be done through the on-device menu.
